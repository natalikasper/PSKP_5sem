Hello World!

Создаём index.js 


const http = require('http');

const server = http.createServer((req, res) => {
    res.end('Hello world!');
});

server.listen(2000, '127.0.0.1', () => {
    console.log('Running');
});

Запускаем через консоль


node index.js

Открываем в браузере


http://127.0.0.1:2000/
